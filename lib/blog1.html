Some Trending JS Frameworks are:-

  1. React JS

  1. REACTJS:-

    React.js or REACTJS is an front-end, open source, javascript library for building user interface or UI component.It is developed by facebook.It can be used as base in development of single page application or mobile application.It is a powerful library that is optimal for fetching data which is rapidly changing.Its main function is data rendering without reloading the page.
    Some Example of applications which built in react.js are as follows:-Facebook,Instagram,Netflix,Yahoo!Mail etc.

      Features:-

    1. Simplicity:- It use JSX file which makes application simple and easy to understnd as well as code.

    2. Open-Source:- It isan open source framework which we can open code and reuse it.

    3. JSX:- stands for Javascript XML.It is an syntax extension to javascript.It allows to write and add HTML in react.

    4. One Data Binding:- One-way binding of data means data can be bind from component to view and in one-direction. As reactjs is based on component based approach so this language is a designed in such manner that follows uni-directional flow or one-way binding.The advantage of one-way binding give you better control throughout the application.

    5. Virtual DOM:-As virtual DOM is like a blueprint.It contains all the details needed to construct the DOM but beacuse it doesnot have heavyweight part that go in real DOM, it can be easily created and change.

    6. Debugging:-React applications are easy to test due to a large developer community. Facebook even provides a small browser extension that makes React debugging faster and easier.

  2. NODEJS:-

    Node.js is am open source, back-end ,javascript runtime environment which executes javascript code outside a web browser.It is server-side scripting language. This scripting language is built by Ryan Dhal in 2009 and its latest version is v0.10.36.This application is also written in javascript.This uses google chrome8 engine under chrominum project.
    Some Examples of applications which are built in node.js are as follows:- Netflix,eBay,Uber etc.

      Features:-

    1. Fast:- Code execution is extermely fast as it is built on V8 javascript google chrome engine.

    2. I/O is asynchronous and event-driven:- All API of node.js library are asynchronous that means here server doesnot wait for api to come back the data.The server call will move to next api oe by one while a notification meachanism of event is generated to response pervious called API's.

    3. Highly-Scalable:-As with event mehanism hel the server to response fast in non-blocking way and makes the server scaleable.

    4. License:- Node.js is released under MIT license.

    5. Non-buffering:-Node.js doenot buffer any data.

  3. ANGULARJS:-

    AngularJs is very powerful javascript framework.It is open source, front-end web framework which is mainly maintained by google.It is used in Single Page Application.AngularJs is developed by Misko Hevery and Adam Abrons. Its latest version is 1.2.21.It extends HTML DOM with additional attributes and makes it more responsive to user actions.
    Some Examples of application which are built in Angular are as follows:- Amazon,Google etc.

      Features:-

    1. MVC(Model-View-Controller):-This framework is built on the most popular concept of MVc(Model-View-Controller). This pattern is based on splitting the business logic layer, the data layer, and presentation layer into separate sections.The division into different sections is done so that each one could be managed more easily.

    2. Data-Binding:-you donot need to write code to bind data to Html controls.So, this done by angukar by adding few snippets to it.

    3. Unit-Testing Ready:-Unit testing means to divide the module in small units of code. So, in Anuglar it first inject and test if the request is returned with required data which ensures that each and every module is as per your requirement.

  4. EXPRESSJS:-

    5. Express.js or express is a back-end web framework for Node.js which is open-source and released under MIT license. It is designed for building web applications and API's.This was foundedd by TJ Holowaychuk.This is written in javascript
    Some Examples of application which are built in Express.js are as follows:-Fox Sports,PayPal,Uer etc.

      Features:-

    1. Faster Server side development:- It provides many common used features of Node.js in the form of functions that can be readily used anywhere in the program. This removes the need to code for several hours and thus saves time.

    2. Routing:- It provides a highly advanced routing mechanism which helps to preserve the state of the webpage with the help of their URLs.

    3. Templating:- ExpressJS provides templating engines that allow the developers to build dynamic content on the web pages by building HTML templates on the server-side.

    4. Middleware:- Middleware is a part of the program that has access to the database, client request, and the other middlewares. It is mainly responsible for the systematic organization of different functions of Express.js.
